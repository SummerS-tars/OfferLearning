# 06

LCR 123. 图书整理 I

## 题目描述

简单来说就是倒序输出链表  

数据规模：  

长度在 [0, 1000] 之间

## 思路

递归  
问题分解：当前节点+后续节点倒序输出  
递归边界：当前节点为空

栈  
将节点依次入栈，最后依次出栈即可

```cpp
class Solution {
public:
    vector<int> reverseBookList(ListNode* head) {
        vector<int> result;
        stack<int> inOrderResult;
        for (auto curNode = head; curNode; curNode = curNode->next)
            inOrderResult.push(curNode->val);
        while(!inOrderResult.empty()) {
            result.push_back(inOrderResult.top());
            inOrderResult.pop();
        }

        return result;
    }
};
```
