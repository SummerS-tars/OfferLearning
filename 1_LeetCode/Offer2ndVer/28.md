# 28

LCR 145. 判断对称二叉树

## 题目描述

root：一棵树  

要求：  
判断这棵树是否轴对称。  

## 思路

适合使用bfs变体  
两个队列同时遍历左右子树，比较节点值是否相等。  

细节处理：  
先初始化两个队列，分别放入根节点的左右子节点。  
匹配？  
获取两个队头，如果均为空，直接跳过  
如果只有一个为空，匹配失败  
如果值不同，匹配失败  
否则将其子节点继续入队。
入队细节？  
左子树从左往右入队，右边则相反，如果碰到空节点也入队（使用空占位）  
最后，如果两个队列均为空，说明匹配成功。  

如果考虑使用递归处理：  
使用一个辅助函数，传入root的左右子节点作为两棵树A, B  

问题分解：  
A与B的根节点值是否相等？  
A的左子树是否与B的右子树对称？  
A的右子树是否与B的左子树对称？  

边界条件：  
A与B均为空，返回True  
A或B有一个为空，返回False  
A与B的值不等，返回False  
否则递归判断子树。
