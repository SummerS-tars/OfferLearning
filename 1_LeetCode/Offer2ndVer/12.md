# 13

LCR 129. 字母迷宫

## 题目描述

grid: m * n 二维字符数组  

target: 目标字符串  

需求：  
按照字母顺序试图在grid中找到target  
输出是否能找到  

限制：  
m == grid.length  
n = grid[i].length  
1 <= m, n <= 6  
1 <= target.length <= 15  
grid 和 target 仅由大小写英文字母组成  

## 思路

直接思路：  

动态规划  

外层：遍历矩阵，找起点  
内层：深度优先搜索（DFS）  
维护一个访问数组，防止重复访问  

实现思路：  

定义四个方向
