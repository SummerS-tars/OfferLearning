# 22

LCR 140. 训练计划 II

## 题目描述

head：头节点  
指向一个链表  
每个链表中有val  

val：int  
训练项目编号  

cnt：int

要求：  
返回倒数第cnt个训练编号  

限制：
1 <= head.length <= 100
0 <= head[i] <= 100
1 <= cnt <= head.length

## 思路

最直接：  
先遍历一边，得到长度  
然后重新遍历一边，找到倒数第cnt个节点  

省时思路：  
开数组，边遍历边存储节点值  

递归思路：  
借助全局量记录长度  

快慢指针：  
快指针先走cnt步  
然后快慢指针一起走，直到快指针到达末尾  

总之思路很多，也比较简单
